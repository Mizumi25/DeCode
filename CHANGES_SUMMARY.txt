TECHNICAL DETAILS:
------------------

OLD APPROACH (V1):
- Tried to clone DOM to offscreen container
- Cloning often failed, frames didn't copy
- Thumbnails were missing or blank
- Complex, unreliable code

NEW APPROACH (V2):
- Read frame data from live DOM
- Manually render each frame using Canvas API
- Draw actual thumbnail images with ctx.drawImage()
- Pixel-perfect, predictable results
- Simple, maintainable code

KEY INNOVATION:
- Instead of taking DOM screenshots, we MANUALLY RENDER each frame
- We draw the frame container (background, border, shadow)
- We draw the frame header (title)
- We draw the ACTUAL thumbnail image (from Playwright)
- Result: Framer-quality high-fidelity previews!
